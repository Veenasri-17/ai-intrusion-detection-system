AI Intrusion Detection System for Cloud Security
📌 Overview

This project is an AI-based Intrusion Detection System (IDS) designed to enhance cloud security by detecting malicious network activities such as DDoS attacks and zero-day threats using machine learning and deep learning techniques.

The system improves detection accuracy using data augmentation and ensemble learning methods and can be integrated into a web-based deployment using Flask.

🎯 Objective

To build an intelligent intrusion detection system capable of identifying and classifying malicious network behavior in cloud environments using advanced machine learning techniques.

⚙️ Features
Detection of DDoS attacks and zero-day threats
Data preprocessing and feature engineering
Data augmentation for improving model performance
Stacking classifier for improved accuracy and robustness
Model evaluation using performance metrics
